# Cara Menggunakan Database Diagram di dbdiagram.io

## Langkah-langkah

1. **Buka website dbdiagram.io**
   - Kunjungi: https://dbdiagram.io

2. **Buat akun atau login** (opsional, untuk menyimpan diagram)

3. **Import file .dbml**
   - Klik tombol **"Import"** atau **"New Project"**
   - Pilih **"Import from .dbml file"**
   - Upload file `fwc_database.dbml`
   
   **ATAU**
   
   - Copy semua isi dari file `fwc_database.dbml`
   - Paste ke editor di dbdiagram.io

4. **Diagram akan otomatis ter-render**
   - Anda akan melihat visual diagram database dengan semua tabel dan relasi

5. **Fitur yang bisa digunakan:**
   - **Zoom in/out** - Perbesar/perkecil diagram
   - **Export** - Export sebagai PNG, PDF, atau SQL
   - **Share** - Bagikan link diagram
   - **Edit** - Edit diagram langsung di browser

## Export Options

Setelah diagram ter-render, Anda bisa export sebagai:

- **PNG** - Gambar diagram
- **PDF** - Dokumen PDF
- **SQL** - Generate SQL DDL script
- **PostgreSQL** - Generate PostgreSQL script
- **MySQL** - Generate MySQL script
- **MSSQL** - Generate SQL Server script

## Tips

- Diagram akan menampilkan semua relasi antar tabel secara visual
- Klik pada tabel untuk melihat detail kolom
- Hover pada relasi untuk melihat foreign key relationship
- Gunakan fitur **"Auto Layout"** untuk mengatur posisi tabel secara otomatis

## File yang Tersedia

- **fwc_database.dbml** - File diagram untuk dbdiagram.io
- **fwc_database.sql** - SQL DDL script (sudah tersedia)
- **DATABASE_DESIGN.md** - Dokumentasi lengkap
- **ERD_DIAGRAM.md** - ERD dalam format Mermaid dan text
